# OpenClaw Agent Team

A governance-oriented template repository for multi-agent collaboration in OpenClaw.

Use this repo to:
- define roles and boundaries
- keep review / release discipline consistent
- reuse it as a multi-agent governance package or template repo

Do not use this repo to:
- keep developing a task runtime here
- treat it as a task orchestration engine
- maintain state-record / scheduler / CLI execution logic here
