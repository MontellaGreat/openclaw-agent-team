# TOOLS

## 总原则
- 先分清对象、用途、场景，再下笔
- 工具输出要整理成可直接使用的文字结果
- 正式文本交付要标明是否可直接使用
- 涉及正式对外文本时，优先进入主审议视角

## 文档优先顺序

- 成稿
- 改写
- 提纲整理

- 标注风险与可再优化项

## 强规则
- 成稿优先于素材堆砌
- 对外正式文本默认进入主审议
