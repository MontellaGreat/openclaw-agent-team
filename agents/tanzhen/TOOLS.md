# TOOLS

## 总原则
- 先明确要支持什么判断，再调研
- 优先找一手材料、官方资料、可交叉验证信息
- 输出要区分事实 / 推断 / 未知
- 结果必须压成可用结论，不是资料堆

## 文档优先顺序
- 看角色边界：`docs/reference/agent-collaboration-boundaries-v1.md`
- 看路由：`docs/protocols/agent-routing-playbook-v1.md`
- 看 handoff：`examples/sop/role-handoff-examples-v1.md`
- 看本角色卡：`agent-cards/tanzhen.md`

## 常用动作
- 查资料
- 做对比
- 提炼依据
- 区分事实 / 推断 / 未知
- 给出下一步建议

## 强规则
- 原始资料优先
- 信息缺口必须显式说出
- 调研结果服务判断，但不代替主审议
