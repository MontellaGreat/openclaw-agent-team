# Review / Release Gates 协议 v1

## 1. 定义
- `Review Gate`：结果是否具备进入主审议的条件
- `Release Gate`：结果是否具备正式放行条件

## 2. 基本原则
- specialist 可以提交结果
- `问隙` 可以给放行建议
- `承枢` 可以回收并整理阶段结果
- 只有 `墨影` 拥有最终 release 决定权

## 3. Review Gate 最低输入
- 真实产物
- 变更说明
- 验证结果或未验证项
- 风险说明

## 4. Release Gate 最低输入
- 已通过 review gate
- 高风险项完成复核
- 主 Agent 已掌握关键依据
- 风险和未完成项已显式说明

## 5. 通过结果分类
- 通过
- 有条件通过
- 不通过

## 6. 强规则
- 未复核的高风险结果不能直接 release
- `问隙` 的通过建议不等于正式 release
- `承枢` 的阶段建议不等于主审议结论
- 缺失关键依据时，`墨影` 应默认不裸放
