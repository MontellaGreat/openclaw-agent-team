# 协议到角色卡映射计划 v1

## 目标
把已经成型的协作协议、路由规则、handoff 要求、review/release gate 原则，反向收进角色卡，使角色卡不只是介绍文，而是可执行入口。

## 本轮映射重点

### 1. 墨影
补强：
- review / release gate 最终权
- 路由决定权
- 有条件通过 / 不通过判断口径

### 2. 承枢
补强：
- handoff 回收格式
- 阶段总结和 gate 前整理职责
- 不越权进入主审议

### 3. 铁手
补强：
- 实现回报格式
- review gate 前最小交付要求
- 高风险结果默认等待复核

### 4. 笔官
补强：
- 内容交付回报格式
- 可直接使用标准
- 对外文本进入主审议前的边界

### 5. 问隙
补强：
- review gate / release gate 建议职责
- 放行建议不等于正式放行
- 未验证项显式标出

### 6. 探针
补强：
- 事实 / 推断 / 未知分层
- 调研结果作为前置输入，不代替主审议

### 7. 观象
补强：
- 巡检观察格式
- 异常信号暴露进入主链的方式

### 8. 片场
补强：
- 多媒体交付规格回报
- 初稿 / 中间版 / 定稿边界
