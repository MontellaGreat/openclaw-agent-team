# Routing Rules

> 本文件是路由规则的速查摘要。
> 若你需要完整版本，请优先阅读 `docs/agent-routing-matrix.md`。

## 默认分发
- 资料检索 / 文档调研 / 方案对比 -> 探针
- 代码 / 脚本 / 自动化 / 配置实现 -> 铁手
- 测试 / 复现 / 边界 / QA / 放行评估 -> 问隙
- 文案 / 文章 / 脚本 / 润色 -> 笔官
- 日常运营 / 例行执行 / structured ops / cron 任务 -> 观象
- 图片 / 视频 / 素材生产 -> 片场
- 多步骤任务编排与状态推进 -> 承枢

## 默认原则
- 能单线就不并行
- 信息不足先补信息
- 高风险任务不直接执行
- 正式内容先草案后发布
- 多步骤任务优先考虑承枢承担编排协调劳动
