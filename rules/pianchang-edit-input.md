# 片场-修图输入规则

来源：用户提供《修图使用说明.md》整理。

---

## 一、默认模型策略

修图任务默认优先按能力选择：
1. `qwen-image-edit-max` / `qwen-image-edit-max-2026-01-16`：复杂编辑、几何推理、角色一致性更强
2. `qwen-image-2.0-pro`：兼顾生成与编辑，适合更通用场景
3. `qwen-image-2.0`：加速版，适合更快响应
4. `qwen-image-edit-plus`：支持多图输出与自定义分辨率
5. `qwen-image-edit`：单图编辑，仅输出 1 张

片场调度时要先判断：
- 是单图编辑还是多图融合
- 是否需要多图输出
- 是否需要明确指定尺寸
- 是否更看重复杂一致性还是速度

---

## 二、适用任务

适用于：
- 改字
- 换背景
- 增删物体
- 调整动作
- 风格迁移
- 提升细节
- 多图融合

不适用于：
- 从零纯文生图（应转片场-生图）
- 视频或动态素材（应转片场-视频）

---

## 三、最小输入要求

片场在分发给片场-修图前，至少要确认：
1. **输入图**：1-3 张
2. **编辑指令 text**：且只能有一个 text
3. **任务类型**：改单图 / 多图融合 / 多图参考编辑
4. **是否需要输出多张图**

建议补齐：
5. `negative_prompt`
6. `size`
7. 是否开启 `prompt_extend`
8. 是否去水印 `watermark=false`
9. 是否要求角色/服装/动作一致性

如果没有输入图，不能当修图任务直接开工。

---

## 四、输入限制

### 图像数量
- 支持输入 1-3 张图
- 多图输入时，顺序有意义
- 多图输入时，输出图比例默认以**最后一张图**为准

### 图像格式
支持：
- JPG / JPEG / PNG / BMP / TIFF / WEBP / GIF

说明：
- GIF 仅处理第一帧
- 单张图片建议宽高在 `384-3072` 像素之间
- 单张大小不超过 `10MB`

### 文本指令
- 只能传一个 `text`
- 支持中英文
- 长度不超过 `800` 字符

---

## 五、可控参数

- `n`：输出张数，通常 1-6（部分模型仅支持 1）
- `negative_prompt`
- `size`
- `prompt_extend`
- `watermark`
- `seed`

### size 规则
- `qwen-image-2.0` 系列：总像素需在 `512*512` 到 `2048*2048` 之间
- `qwen-image-edit-max / plus`：宽高均在 `[512, 2048]`
- `qwen-image-edit`：不可指定 size

说明：
- 若指定 `size`，系统会自动调整到最接近的 16 倍数尺寸

---

## 六、调度规则

### 规则 1：单图编辑
若用户给 1 张图并要求局部改动：
- 默认走片场-修图
- 模型优先 `qwen-image-edit-max` 或 `qwen-image-2.0-pro`

### 规则 2：多图融合
若用户给多张图并要求合成：
- 明确图 1 / 图 2 / 图 3 的角色与顺序
- 片场必须把顺序写进执行摘要

### 规则 3：需要多结果时明确 n
如果用户要多个方案图：
- 明确 `n`
- 回传时说明实际生成张数

### 规则 4：复杂高要求场景优先 max
涉及：
- 工业设计
- 几何推理
- 角色一致性
- 复杂动作迁移
优先 `qwen-image-edit-max`

### 规则 5：修图结果也有时效
返回图像 URL 可能 **24 小时有效**，片场回传时必须提醒保存。

### 规则 6：地域与密钥不能混用
北京和新加坡地域 API Key 与请求地址不能混用。片场调度前要检查环境，不得盲猜。

---

## 七、片场-修图标准输出

1. 修图目标
2. 输入图数量与角色说明
3. 所选模型
4. 编辑指令摘要
5. 关键参数（n / size / negative_prompt / prompt_extend）
6. 产物说明
7. 风险与限制
8. 保存提醒
